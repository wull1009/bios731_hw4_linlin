# BIOS 731 Homework 4

This repository contains code and analysis for BIOS 731 Homework 4.

## Structure
- `analysis/`: R Markdown report
- `source/`: R scripts for Problems 1–4
- `slurm/`: cluster submission scripts for Problem 4
- `results/`: output files
- `logs/`: cluster log files

## Notes
Problem 4 is designed to run on the cluster using SLURM.
